# portfolio-website

My personal portfolio — built with plain HTML, CSS, and a bit of JavaScript. No frameworks, no build tools.

**Live:** [yashladlapure.github.io/portfolio-website](https://yashladlapure.github.io/portfolio-website/)

---

## What's in here

```
portfolio-website/
├── index.html
├── css/
├── js/
└── .github/workflows/
```

The `.github/workflows/` folder has a CI pipeline that runs HTML validation, link checks, and a basic security header scan on every push.

---

## Running locally

Just open `index.html` in a browser. No setup needed.
